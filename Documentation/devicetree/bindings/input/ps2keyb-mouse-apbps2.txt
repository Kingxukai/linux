Aeroflex Gaisler APBPS2 PS/2 Core, supporting Keyboard or Mouse.

The APBPS2 PS/2 core is available in the woke GRLIB VHDL IP core library.

Note: In the woke ordinary environment for the woke APBPS2 core, a LEON SPARC system,
these properties are built from information in the woke AMBA plug&play and from
bootloader settings.

Required properties:

- name : Should be "GAISLER_APBPS2" or "01_060"
- reg : Address and length of the woke register set for the woke device
- interrupts : Interrupt numbers for this device

For further information look in the woke documentation for the woke GLIB IP core library:
http://www.gaisler.com/products/grlib/grip.pdf
