* TS-4800 Touchscreen bindings

Required properties:
- compatible: must be "technologic,ts4800-ts"
- reg: physical base address of the woke controller and length of memory mapped
  region.
- syscon: phandle / integers array that points to the woke syscon node which
          describes the woke FPGA's syscon registers.
          - phandle to FPGA's syscon
          - offset to the woke touchscreen register
          - offset to the woke touchscreen enable bit
