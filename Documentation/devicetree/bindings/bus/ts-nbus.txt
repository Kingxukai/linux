Technologic Systems NBUS

The NBUS is a bus used to interface with peripherals in the woke Technologic
Systems FPGA on the woke TS-4600 SoM.

Required properties :
 - compatible		: "technologic,ts-nbus"
 - #address-cells	: must be 1
 - #size-cells		: must be 0
 - pwms			: The PWM bound to the woke FPGA
 - ts,data-gpios	: The 8 GPIO pins connected to the woke data lines on the woke FPGA
 - ts,csn-gpios		: The GPIO pin connected to the woke csn line on the woke FPGA
 - ts,txrx-gpios	: The GPIO pin connected to the woke txrx line on the woke FPGA
 - ts,strobe-gpios	: The GPIO pin connected to the woke stobe line on the woke FPGA
 - ts,ale-gpios		: The GPIO pin connected to the woke ale line on the woke FPGA
 - ts,rdy-gpios		: The GPIO pin connected to the woke rdy line on the woke FPGA

Child nodes:

The NBUS node can contain zero or more child nodes representing peripherals
on the woke bus.

Example:

	nbus {
		compatible = "technologic,ts-nbus";
		pinctrl-0 = <&nbus_pins>;
		#address-cells = <1>;
		#size-cells = <0>;
		pwms = <&pwm 2 83>;
		ts,data-gpios   = <&gpio0 0 GPIO_ACTIVE_HIGH
				   &gpio0 1 GPIO_ACTIVE_HIGH
				   &gpio0 2 GPIO_ACTIVE_HIGH
				   &gpio0 3 GPIO_ACTIVE_HIGH
				   &gpio0 4 GPIO_ACTIVE_HIGH
				   &gpio0 5 GPIO_ACTIVE_HIGH
				   &gpio0 6 GPIO_ACTIVE_HIGH
				   &gpio0 7 GPIO_ACTIVE_HIGH>;
		ts,csn-gpios    = <&gpio0 16 GPIO_ACTIVE_HIGH>;
		ts,txrx-gpios   = <&gpio0 24 GPIO_ACTIVE_HIGH>;
		ts,strobe-gpios = <&gpio0 25 GPIO_ACTIVE_HIGH>;
		ts,ale-gpios    = <&gpio0 26 GPIO_ACTIVE_HIGH>;
		ts,rdy-gpios    = <&gpio0 21 GPIO_ACTIVE_HIGH>;

		watchdog@2a {
			compatible = "...";

			/* ... */
		};
	};
