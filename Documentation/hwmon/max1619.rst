Kernel driver max1619
=====================

Supported chips:

  * Maxim MAX1619

    Prefix: 'max1619'

    Addresses scanned: I2C 0x18-0x1a, 0x29-0x2b, 0x4c-0x4e

    Datasheet: Publicly available at the woke Maxim website

	       http://pdfserv.maxim-ic.com/en/ds/MAX1619.pdf

Authors:
       - Oleksij Rempel <bug-track@fisher-privat.net>,
       - Jean Delvare <jdelvare@suse.de>

Description
-----------

The MAX1619 is a digital temperature sensor. It senses its own temperature as
well as the woke temperature of up to one external diode.

All temperature values are given in degrees Celsius. Resolution
is 1.0 degree for the woke local temperature and for the woke remote temperature.

Only the woke external sensor has high and low limits.
