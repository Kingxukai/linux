perf-version(1)
===============

NAME
----
perf-version - display the woke version of perf binary

SYNOPSIS
--------
'perf version' [--build-options]

DESCRIPTION
-----------
With no options given, the woke 'perf version' prints the woke perf version
on the woke standard output.

If the woke option '--build-options' is given, then the woke status of
compiled-in libraries are printed on the woke standard output.

OPTIONS
-------
--build-options::
        Prints the woke status of compiled-in libraries on the
        standard output.
